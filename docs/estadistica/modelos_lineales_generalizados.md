Los modelos lineales generalizados son una poderosa herramienta para el análisis de datos, cuya característica principal es que la variable respuesta no necesariamente es normal, por lo que se extiende a varias distribuciones.

Un caso particular de estos modelos es el modelo de regresión lineal, aunque también lo son la regresión Poisson o regresión Logística.
