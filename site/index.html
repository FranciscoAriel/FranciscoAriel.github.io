    SAS ® Software¶

    En este blog conoceras todo lo relacionado a:

    • SAS ® software - Contiene todo lo esencial para leer, manipular datos así como otras funciones básicas de SAS.
    • SAS/STAT ® software - Tiene muchos métodos estadísticos y otros análisis de datos.
    • SAS/IML ® software - Es un lenguaje específico para trabajar con vectores, matrices, funciones matemáticas y estadísticas avanzadas.
    • OTROS - Conexión de SAS con otros lenguajes de programación y sus configuraciones.
