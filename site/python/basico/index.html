    Librerías¶

    El primer paso es llamar las librerías que se utilizarán en la sesión. Usaremos pandas y numpy y otra librería adicional llamada os.

    1
2
3
    import pandas as pd
import numpy as np
import os

    Nota

    En ocasiones es preferible usar nombres más cortos para referirnos a los nombres de las librerías. Por ejemplo, a pandas le llamaremos pd y np para referirnos a numpy.

    Ahora llamaremos a la función para leer los datos.

    1
    datos = pd.read_csv("census.csv")

    Como puede observarse, datos es un objeto DataFrame que se define gracias a la función read_csv() de pandas. Se puede obtener información del objeto usando el siguiente comando.

    1
    datos.info()

    Para ver un resumen de los datos se puede usar

    1
    datos.describe()

    Para visualizar los primeros 5 datos se puede usar este código

    1
    datos.head(n=5)
